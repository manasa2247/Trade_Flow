# Trade_Flow – Stock Trading Platform

## Overview

TradeFlow is a stock trading platform inspired by modern brokerage applications like Zerodha.
It allows users to track stocks, view market data, and simulate trading operations through a clean and interactive interface.

## Features

*  View stock market data
*  Track price movements and trends
*  Simulated buy/sell functionality
*  Portfolio management
*  Search and explore stocks
